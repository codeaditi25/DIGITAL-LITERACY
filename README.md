1.  Project Overview
This repository contains the complete Digital Literacy Portfolio submitted as part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The project is built around the scenario of being appointed as a Student Digital Ambassador – helping batchmates navigate the digital world safely and professionally.
The portfolio spans five tasks (one per course module) and is organised in a single public GitHub repository. All deliverables – infographic, profile screenshots, coding challenge evidence, email drafts, checklists, and the full project report – are stored in clearly named folders as specified in the project brief.
No prior coding knowledge was required. Every tool used is free and browser-based.

3.  Module Summaries
Task 1 – Digital Literacy Awareness Infographic  [Module 1 | 20 Marks]
A one-page infographic was created using Canva covering three core digital literacy topics: the definition of digital literacy, safe internet practices, and maintaining a professional online presence. The design uses a colour-coded layout with icons and concise bullet points for quick readability. The completed infographic was exported as a PNG file.
Tool used: Canva (canva.com)
Topics covered: Digital literacy definition  |  Safe internet practices  |  Professional online presence
Deliverable: task-1-presentation.png
Task 2 – Student Digital Portfolio  [Module 2 | 20 Marks]
Professional profiles were set up on three platforms: GitHub, LinkedIn, and Kaggle. On GitHub, both a project repository and a profile README were created. On LinkedIn, the Education section was completed. On Kaggle, a profile was created and free learning courses were explored.
Platforms: GitHub  |  LinkedIn  |  Kaggle
GitHub repo: https://github.com/codeaditi25/DIGITAL-LITERACY
Deliverable: task-2-portfolio/ (3 screenshots)
Task 3 – Coding & Collaboration Platforms  [Module 3 | 20 Marks]
Part A: HackerRank – two beginner challenges completed ('Solve Me First' and 'Python: Division'), both scoring 10/10. Part B: A five-question Digital Literacy Awareness Quiz was built using Google Forms (3 multiple-choice + 2 short-answer), linked to a Google Sheet that auto-collects responses.
Part A: HackerRank – Solve Me First + Python Division  (both 10/10)
Part B: Google Forms – 5-question Digital Literacy Awareness Quiz
Form link: https://docs.google.com/forms/d/e/1FAIpQLSfZ-tmO745YmDUJdbHhuYCG8CL2OeqoZSIFha2YK4VcOETu1g/viewform?usp=header
Deliverable: task-3-platforms/ (3 screenshots)
Task 4 – Email Etiquette & Professional Communication  [Module 4 | 20 Marks]
Part A: Two formal emails were drafted – (1) a student requesting an assignment extension from a professor, and (2) a student expressing interest in a summer internship. Both follow professional conventions. Part B: A Social Media Do's and Don'ts checklist was created with 7 Do's and 7 Don'ts for responsible digital communication.
Deliverable: task-4-email-etiquette/ (2 files)
Task 5 – Cybercrime Awareness Case Study  [Module 5 | 20 Marks]
Part A: A 200–250 word case study on UPI/Online Payment Fraud – specifically the 'QR Code Payment Request' scam – written as a realistic fictional scenario involving a first-year student selling a laptop online. Part B: A 13-point 'Stay Safe Online' checklist including 2 UPI-specific tips and official reporting channels.
Cybercrime type: UPI / Online Payment Fraud
Report channels: cybercrime.gov.in  |  Helpline: 1930 (24x7)
Deliverable: task-5-cybercrime/ (2 files)
